# Chet.Admin 项目文档

> Chet.Admin 是一套基于 **Vben Admin v5.7 + .NET 10 WebAPI** 的全栈 RBAC 权限管理系统，采用前后端分离架构。

本目录包含整个项目（前端 + 后端）的完整技术文档，面向项目开发者、维护者和部署人员。

## 项目结构概览

```
Chet.Admin/
├── Chet.Admin.Api/          # 后端 .NET 10 解决方案（Clean Architecture）
│   ├── Chet.Admin.Api/              # 表示层：控制器、中间件、启动配置
│   ├── Chet.Admin.Application/      # 应用层：业务服务、对象映射
│   ├── Chet.Admin.Core/             # 核心层：领域模型、接口契约、共享类
│   ├── Chet.Admin.Infrastructure/   # 基础设施层：数据访问、缓存、配置、日志
│   └── Chet.Admin.Tests/            # 测试层：单元测试、集成测试
├── Chet.Admin.Web/          # 前端 Vben Admin 5.7 Monorepo
│   └── apps/web-antd/               # 主应用（Ant Design Vue）
└── docs/                    # 项目文档（本目录）
```
